# CadastroDinamicoWeb
Ferramenta em ASP.NET Core para realizar operações CRUD dinâmicamente em banco de dados SQL Server.

## Introdução

Essa ferramenta é capaz de se conectar a qualquer servidor de banco de dados SQL Server, mapear suas databases, schemas, tabelas, colunas, chaves primárias, estrangeiras, entre outras coisas.
Com essas informações, serão geradas telas dinamicamente para a realização das operações:
* Create
* Read
* Update
* Delete

O usuário e senha padrão da aplicação é:
* Usuário: admin
* Senha: admin!@#123
